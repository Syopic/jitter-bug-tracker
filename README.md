## About Jitter

Jitter is an immersive sci-fi exploration game that takes you on an adventure through mysterious worlds. This repository is dedicated to tracking bugs, issues, and feature requests to help us make the game better for everyone.

## ğŸ› How to Report a Bug

If you've encountered a bug or issue while playing Jitter, please help us by reporting it! 

1. **Check existing issues** - Search the [issues page](../../issues) to see if someone has already reported the same problem
2. **Create a new issue** - If your issue is new, click on [New Issue](../../issues/new/choose) and select the appropriate template
3. **Fill out the template** - Provide as much detail as possible to help us understand and reproduce the issue

### What to Include in Your Bug Report

- **Clear description** of what went wrong
- **Steps to reproduce** the issue
- **Expected vs. actual behavior**
- **Game version** (you can find this in the game menu)
- **Platform** (Windows, Linux, macOS, Steam Deck)
- **System specifications** (CPU, GPU, RAM)
- **Screenshots or videos** if applicable
- **Log files or error messages** if available

Issues are organized using labels to help track their status:

- `bug` - Something isn't working correctly
- `enhancement` - New feature or improvement suggestion
- `question` - Questions about the game
- `needs-triage` - New issues that need to be reviewed
- `needs-review` - Issues that need developer review
- `duplicate` - This issue already exists
- `invalid` - Issue is not valid or not a bug
- `wontfix` - This will not be worked on
